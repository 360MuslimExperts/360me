## 2024-05-22 — Image Loading Optimization in Blog
**Learning:** Found that the blog page, which acts as a primary landing for insights, was missing 'priority' on the featured post and responsive 'sizes' on all post images. This leads to poor LCP and oversized image requests on mobile.
**Action:** Implement 'priority' for the first item in the map and use responsive 'sizes' reflecting the grid layout (1, 2, or 3 columns).
